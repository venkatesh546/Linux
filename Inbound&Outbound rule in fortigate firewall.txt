Create NAT Inbound rule
Step 1 goto policy and object > IPv4 Policy 
click Virtual Ips > create New 
Name  = Anything you want
Comments = any
Under Network
Interface  = select external wan 
Type  =  default
External IP Address/Range = public IP 
Mapped IP Address/Range  =  internal IP
Optional Filters = Disabled
Port Forwarding = enable
Protocal = TCP
External Service Port =  specific port
Map to port =  custom port

click OK

And Navigate to IPv4 Policy > Create New
Name  = Anything you want
Incoming Interface =  External Interface(WAN IP)
Outgoing Interface =  Internal LAN
Source    = all
Destination = Virtual Ip
Schedule = always
Service = All
Action = Accept

NAT = unchecked

Click OK
----------------------------------------------------
