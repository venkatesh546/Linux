TO CONFIGURE THE MYSQL DATABASE ######
Step 1  :	TO INSTALL THE  PACKAGE ######

yum  groupinstall  *MYSQL*  -y

rpm -qa  mysql*  ( to query  the  pkgs )

Step2 TO START THE   SERVICE #####

service  mysqld  restart

Step 3 TO ASSIGN THE PASSWORD FOR MYSQL DATABASE ####

mysqladmin -u  root  password  redhat

Step 4 TO LOGIN INTO MYSQL DATABASE #####

mysql -u root -p

password: assign the password 

Step 5  TO SHOW THE MYSQL DATABASE #########
mysql> show databases;

+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| test               |
+--------------------+
3 rows in set (0.00 sec)

step 6  TO  CREATE A DATABASE ######
mysql>creat database zoom;

Query OK, 1 row affected (0.00 sec)

Step 7 TO  CHECK THE DATABASE ######

mysql> show databases;

Step 8 TO ENTER THE DATABASE  ######

mysql> use zoom;

Step 9  TO DISPLAY THE DATABASE TABLE ####
mysql> show tables;


Empty set (0.00 sec)



Step 10 TO  CREATE A TABLE WITH FIELDS #######

mysql> create table linux (Name varchar(20) not null, Phone varchar(10), Status varchar(10));



Query OK, 0 rows affected (0.07 sec)

Step 11 TO SHOW THE TABLE ######

mysql> show tables;

+----------------+

+----------------+
